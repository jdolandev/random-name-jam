Random Name Jam

Making games starting with nothing but a random name!

What?

Recently, videogamena.me made some buzz on Twitter. So @pstn_, @peterschueler and @lnwdr came up with the idea of making actual games out of the random names from that site.

Here are a few examples of what awaits you:

  • Supreme Quantum Trivia
  • Robot Gimp Mission
  • Return of Disco Fantasy
  • Vegetarian Dating Thieves

And of course, everyone is welcome to join us!

Where?

Where ever you want. To keep organization simple, we don't have an actual meetup planned.

When?

From 2015-03-23 00:00:00 to 2015-03-29 23:59:59.

iCal

The Rules

Anyone interested in participating should add their Github username to the list below via pull request.

Hit the EDIT button on this page, copy the empty list entry and create a pull request.

Alternatively, you can also contact @lnwdr on Twitter (or any other communication channel) to join.

When the jam starts, you will be given 3 random game names, from which you pick one, and develop a game with that name. The name will be posted under your entry on the participants list.

You can pick any platform you like but of course it'd be nice, if as many people as possible can actually play your game.

When the deadline comes, there will be a new list here for the finished games. Again, add yours via pull request (or other means of communication), just as before.

This is not a contest. There will be no winner, no prize, just games.

For questions and discussions we can use the issue tracker.
